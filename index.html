Dijkstra's Algorithm Illustrated

2020 • Interactive tool

A third desktop game, Dijkstra's Algorithm is known as one of the fastest search algorithms for certain types of graphs. In this instructive game, a player uses the arrow key to move an image around the screen, with a wrap-around board, while Dijkstra's Algorithm, highlighted in yellow, directs the adversary toward the player's position. The game ends when the adversary reaches the player. The code is plain JavaScript, with object oriented design.
